# Workshop Angular

Le but de ce Workshop est de créer une web app simple a l'aide du framwork Angular.
Dans ce workshop nous aborderons plusieurs notions importantes et nécéssaire pour coder en Angular.

## Installation

Vous pouvez clone le repo avec ce lien : [](https://github.com/ZeroJnr/Workshop-Angular.git)pour vous rendre au dernier Workshop codé.

```bash
git clone https://github.com/ZeroJnr/Workshop-Angular.git
```

## Usage
Pour l'utilisation du projet vous pouvez lancer le script ```./script.sh [WORKSHOP]``` Pour installer automatiquement les dependances et ainsi rejoindre le Workshop.
Il installera NPM et Boostrap4
